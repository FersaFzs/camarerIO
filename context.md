# Contexto del Proyecto: camarerIO

## Objetivo Principal
Mantener la comunicación y sincronización entre el entorno de hardware local (como impresoras térmicas, red local en el restaurante) y la aplicación en la nube desplegada en Render (backend, base de datos).

## Hardware Local
- Gestionar dispositivos físicos en el local.
- Asegurar funcionamiento incluso con cortes de internet temporales.

## Render
- Backend y base de datos accesibles globalmente.
- Sincronización de pedidos y configuraciones.
